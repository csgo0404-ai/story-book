# Cloudflare Workers + D1 배포

## 1. wrangler 설치 & 로그인 (PowerShell)

```powershell
npm install -g wrangler
wrangler login
```

브라우저가 열리면 Cloudflare 계정으로 인증.

## 2. 이 폴더로 이동

```powershell
cd "d:\TO DO LIST\☆코딩\cloudflare-backend"
```

## 3. D1 데이터베이스 생성

```powershell
wrangler d1 create donghwa
```

출력에 나오는 `database_id = "..."` 값을 복사해서
**wrangler.toml**의 `PASTE_DATABASE_ID_HERE` 자리에 붙여넣으세요.

## 4. 스키마 적용

```powershell
wrangler d1 execute donghwa --remote --file=./schema.sql
```

## 5. 배포

```powershell
wrangler deploy
```

배포 완료 후 출력되는 URL을 복사 (예: `https://donghwa-api.<계정>.workers.dev`).

## 6. HTML 수정

`동화책 제작.html` 773번 줄의 `API_URL`을 위에서 받은 Worker URL로 교체:

```js
const API_URL = 'https://donghwa-api.<계정>.workers.dev';
```

끝. 이미지(Cloudinary), 비밀번호(csgo10) 모두 그대로 동작합니다.

## 비밀번호 변경하려면
`wrangler.toml`의 `TEACHER_PW` 값 수정 후 `wrangler deploy` 다시 실행.

## 로그 확인 / 디버깅
```powershell
wrangler tail
```
